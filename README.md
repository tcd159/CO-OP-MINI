# CO-OP-MINI
We are planning to develop mini-games of two-person cooperative games into ai.

## Introduction
This project is a collection of 2-player co-op mini-games designed to test teamwork, communication, and reflexes. Built for two players sharing the same keyboard (or using controllers), this arcade-style bundle requires both players to work together to solve puzzles, survive obstacles, and clear stages.

## Key Features
Dual-Player Synergy: Mechanics specifically designed for two players, where success is impossible alone (e.g., tether mechanics, button-and-gate puzzles).

Diverse Mini-Game Selection: Includes multiple game modes such as "Escape the Maze," "Sync Run," and "Defend the Core."

Shared-Screen Control: Optimized for local multiplayer using a split keyboard layout (WASD + Arrow keys) or dual gamepads.

Dynamic Difficulty: Stages that adapt or scale in difficulty as players progress and improve their teamwork.

## Download

Clone this repository to your local machine:

npm install # or pip install -r requirements.txt (depending on your tech stack)
